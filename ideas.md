# 设计理念：赛博朋克高级感作品集网站

## 选定设计方案：Deep Dark Cyberpunk + Apple Minimalism Fusion

### 设计哲学
融合**赛博朋克的未来感与科技感**与**Apple 的极简优雅**，打造一个既具有高级科技质感，又不失简洁大气的国际化作品集。核心是"**高级感的克制**"——不是堆砌效果，而是通过精细的细节、精准的配色和流畅的交互来传达专业性。

---

## 核心设计原则

### 1. **极简主义 + 科技感**
- 大量留白，让内容呼吸
- 深色背景（几乎纯黑 #0a0a0a）作为主色调
- 极少使用装饰性元素，每个视觉元素都有目的
- 字体选择：清晰、现代、国际化（Geist Sans / Poppins）

### 2. **霓虹紫呼吸灯作为灵魂**
- 主色：紫色 (#a855f7 / #9333ea)，作为强调色
- 呼吸灯效果：用于卡片边框、按钮、交互元素
- 不是满屏霓虹，而是**战略性的点缀**
- 创造"赛博朋克但不仅限于赛博朋克"的高级感

### 3. **液态玻璃 (Glassmorphism) 效果**
- 半透明卡片：`background: rgba(255, 255, 255, 0.05)` + `backdrop-filter: blur(10px)`
- 用于重点内容区块（经验、学历、能力）
- 边框：细微的白色/紫色边框 `rgba(255, 255, 255, 0.1)` 或紫色呼吸灯
- 创造深度感和高级感

### 4. **视差滚动 + 微交互**
- 英雄区域背景缓慢滚动
- 卡片在视口进入时淡入并轻微上升
- Hover 时卡片提升（transform: translateY）+ 边框呼吸灯激活
- 按钮 Hover：背景渐变 + 紫色光晕

### 5. **Bento Box 布局的项目展示**
- 不规则网格布局（3-4 列）
- 每个项目卡片大小不一，创造视觉动态感
- 卡片内容：项目名称、描述、技术栈、链接
- 深色背景 + 紫色呼吸灯边框 + 液态玻璃效果

## 布局范式

### 页面结构
1. **Hero Section**（英雄区）
   - 全屏或 80vh 高度
   - 背景：深色 + 视差效果 + 可选的抽象几何图案
   - 内容：名字、职位、简短介绍、CTA 按钮
   - 文本对齐：左对齐或中心对齐（待定）

2. **About Section**（关于）
   - 液态玻璃卡片
   - 简短的个人介绍

3. **Experience Section**（工作经验）
   - 时间线或卡片列表
   - 每项带有公司名、职位、时间、描述

4. **Education Section**（学历背景）
   - 类似经验的卡片列表

5. **Skills Section**（个人能力）
   - 技能标签云或分类卡片

6. **Projects Section**（项目经历）
   - **Bento Box 不规则网格**
   - 项目卡片大小不一（1x1、2x1、1x2 等）
   - 每卡片：项目名、描述、技术栈、链接

7. **Footer**（页脚）
   - 社交链接、联系方式、版权信息

## 标志性视觉元素

### 1. **紫色呼吸灯边框**
```css
/* 呼吸灯动画 */
@keyframes breathing {
  0%, 100% { box-shadow: 0 0 20px rgba(168, 85, 247, 0.5); }
  50% { box-shadow: 0 0 40px rgba(168, 85, 247, 0.8); }
}
```
- 用于卡片、按钮、重点内容
- 不是持续动画，而是 Hover 时激活或轻微持续

### 2. **液态玻璃卡片**
```css
background: rgba(255, 255, 255, 0.05);
backdrop-filter: blur(10px);
border: 1px solid rgba(255, 255, 255, 0.1);
border-radius: 12px;
```

### 3. **渐变文本**（可选）
- 标题可使用紫色渐变：`background: linear-gradient(to right, #a855f7, #ec4899);`
- 用于强调重要标题

### 4. **几何装饰**（可选）
- 页面背景可有细微的网格或点阵图案
- 不应过于明显，保持简洁

## 交互哲学

### 1. **Hover 效果**
- 卡片：轻微上升 (`transform: translateY(-4px)`) + 呼吸灯激活
- 按钮：背景色变化 + 紫色光晕
- 链接：下划线淡入 + 紫色渐变

### 2. **滚动动画**
- 元素进入视口时：淡入 + 轻微上升
- 背景视差：英雄区背景缓慢滚动
- 不应过度，保持专业感

### 3. **页面过渡**
- 路由切换时的淡入淡出
- 不使用过于花哨的动画

## 排版系统

### 字体选择
- **显示字体**：Geist Sans Bold / Poppins Bold（标题、大标题）
- **正文字体**：Geist Sans Regular / Poppins Regular（正文、描述）
- **等宽字体**：Fira Code（代码、技术栈）

### 字体层级
| 用途 | 大小 | 粗细 | 行高 |
|------|------|------|------|
| H1 标题 | 48-64px | Bold (700) | 1.2 |
| H2 小标题 | 32-40px | Bold (700) | 1.3 |
| H3 副标题 | 24-28px | SemiBold (600) | 1.4 |
| 正文 | 16-18px | Regular (400) | 1.6 |
| 小文本 | 12-14px | Regular (400) | 1.5 |

---

## 总结

这个设计方案的核心是**"克制的科技感"**——通过精细的细节（呼吸灯、液态玻璃、视差）和精准的配色（深黑 + 紫色）来传达高级感，而不是堆砌效果。最终效果应该是：**看起来很专业、很未来、很高级，但又不显得浮躁或过度设计**。
